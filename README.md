# دليل استخدام وتعديل موقع JABI

تهانينا! لقد تم تطوير موقع JABI باستخدام **HTML و CSS و JavaScript** فقط، مما يجعله مفتوح المصدر وسهل التعديل والتحكم الكامل.

## 1. محتويات الملف المضغوط

الملف المضغوط يحتوي على المجلد `jabi_website_final`، ويتضمن الملفات التالية:

1.  `index.html`: ملف الهيكل والمحتوى الرئيسي للصفحة الرئيسية (يشمل Schema Markup).
2.  `blog.html`: صفحة المدونة الرئيسية.
3.  `thank-you.html`: صفحة الشكر المخصصة بعد إرسال النماذج.
4.  `styles.css`: ملف التنسيقات والألوان والتصميم المتجاوب.
5.  `script.js`: ملف التأثيرات الحركية والتفاعلية.
6.  `manifest.json`: ملف تعريف تطبيق الويب التقدمي (PWA).
7.  `service-worker.js`: ملف عامل الخدمة لتمكين التخزين المؤقت والعمل دون اتصال.
8.  `README.md`: هذا الملف (دليل الاستخدام).

## 2. كيفية فتح وتعديل الموقع (باستخدام VS Code)

1.  **فك الضغط:** قم بفك ضغط الملف.
2.  **فتح المشروع:** افتح برنامج **Visual Studio Code (VS Code)**.
3.  **فتح المجلد:** اذهب إلى `File` ثم `Open Folder` واختر مجلد `jabi_website_final`.
4.  **المعاينة:** انقر بزر الماوس الأيمن على ملف `index.html` واختر `Open with Live Server` أو افتحه مباشرة في متصفحك.

## 3. إرشادات التعديل الرئيسية

### أ. تعديل التسعيرات (في `index.html`)

لإجراء تعديلات على الأسعار المقترحة (1,500 ر.س و 3,750 ر.س)، ابحث عن الأسطر التالية في ملف `index.html` وقم بتغيير الأرقام حسب سوقك المحلي:

```html
<!-- الباقة الأساسية -->
<span class="amount">1,500</span>

<!-- الباقة المتقدمة -->
<span class="amount">3,750</span>
```

### ب. تغيير الألوان (في `styles.css`)

تم استخدام متغيرات CSS لتسهيل تغيير الألوان. يمكنك تغيير الألوان الهادئة التي تم اختيارها بسهولة من بداية ملف `styles.css` في قسم `:root`.

### ج. أزرار التواصل العائمة (Floating Contact Buttons)

لتغيير أرقام الهاتف والواتساب، قم بالتعديل في ملف `index.html` و `blog.html` في قسم `floating-contact` (ابحث عن `tel:` و `wa.me/`).

## 4. الميزات الاحترافية المضافة حديثاً

تم إضافة الميزات التالية لزيادة احترافية الموقع:

*   **تطبيق ويب تقدمي (PWA):** يمكن تثبيت الموقع كتطبيق على الهواتف (Android و iOS).
*   **شاشة تحميل (Preloader):** تظهر أثناء تحميل الصفحة.
*   **زر الوضع الداكن:** تم نقله إلى شريط التنقل العلوي.
*   **قسم الأسئلة الشائعة التفاعلي (FAQ Accordion):** قسم جديد في الصفحة الرئيسية يفتح ويغلق بأسلوب احترافي.
*   **صفحة "شكراً لك" مخصصة:** يتم تحويل المستخدم إليها بعد إرسال أي نموذج تواصل.
*   **رسائل منبثقة (Toast Notifications):** رسائل نجاح أنيقة تظهر في زاوية الشاشة.
*   **تأثير تمرير العناصر (Scroll Reveal):** تأثير حركي احترافي لظهور العناصر عند التمرير إليها.
*   **تحسين SEO الهيكلي (Schema Markup):** إضافة بيانات منظمة (JSON-LD) في ملف `index.html` لتحسين ظهور الموقع في نتائج بحث جوجل.
